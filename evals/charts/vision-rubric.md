# Birch chart artifact vision rubric

Validate the generated artifact from desktop and mobile screenshots.

Score each category 1–5:

1. **Birch visual fit**
   - ivory/slate/clay/olive palette
   - calm engineering-report feel
   - no dashboard/glossy SaaS drift

2. **Evidence-first layout**
   - first viewport shows actual numbers or chart evidence
   - hero is compact
   - reader can identify the key finding quickly

3. **Chart readability**
   - chart is legible
   - axes/ticks/labels are readable
   - legend or annotations clarify the series
   - no cramped labels or clipped content

4. **Analytical honesty**
   - chart type matches data
   - exact values appear near visuals
   - values are not communicated by color alone
   - source/caveat is visible

5. **Birch component use**
   - uses cards/panels/metric rows/tables coherently
   - spacing and alignment are stable
   - no obvious custom one-off visual system

6. **Mobile/responsive quality**
   - no horizontal overflow for main meaning
   - chart remains understandable
   - table/card content does not collide

Return scores, pass/fail recommendation, top issues, and suggested improvements.
