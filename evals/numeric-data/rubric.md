# Numeric data rubric

Score each category 1–5.

1. **Evidence-first layout** — first viewport contains actual numbers/chart evidence; hero is compact.
2. **Data understanding** — identifies measures, dimensions, weeks, artifacts, denominators, and caveats correctly.
3. **Chart quality** — trend chart matches the data, has readable axes/legend, and works on mobile.
4. **Exact values** — includes a nearby numeric table with exact values and clear units.
5. **Birch contract** — uses Birch primitives/tokens, no invented design system, no external chart library.
6. **Analytical honesty** — no overstated claims, source/caveat included, values not communicated by color alone.

Birch expected primitives: `.stat-card` or `.card` + `.stat-value`, `.metric-list`, `.metric-row`, `.meter`, `.chart-panel`, `.chart-svg`, `.numeric-table-wrap`, `.numeric-table`, `.callout` or `.card` for caveats.
